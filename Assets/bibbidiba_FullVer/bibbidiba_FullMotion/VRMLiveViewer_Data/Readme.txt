
VMD Retargetを使用する際、モーション付けを行った元モデルのボーン角度情報が必要になります
本来ならば"learningSourcePMX.exe"で学習させる為に必要になりますが
元のモデルを用意できない方がいるかもしれませんので"srcpose.ini"を同梱しておきます。

分かる人だけ使ってください。


--------------------------------------------------------------------------------------------------------------------

When using VMD Retarget, the bone angle information of the original model to which motion was applied is required.
This information is required for “learningSourcePMX.exe” to learn the motion.
However, there may be people who cannot prepare the original model, so “srcpose.ini” is included in the package.

Please use it only if you know how to use it.


--------------------------------------------------------------------------------------------------------------------

使用 VMD Retarget 时，需要附加运动的原始模型的骨骼角度信息。
最初，使用 “learningSourcePMX.exe ”学习时需要这些信息，但有些人可能无法准备原始模型，因此包含了 “srcpose.ini”。

请在理解的前提下使用。


--------------------------------------------------------------------------------------------------------------------

VMD Retarget을 사용할 때, 모션이 적용된 원본 모델의 본 각도 정보가 필요합니다.
원래는 “learningSourcePMX.exe”에서 학습시키기 위해 필요한 정보이지만
원본 모델을 준비하지 못한 분들을 위해 “srcpose.ini”를 함께 제공합니다.

아는 사람만 사용하시기 바랍니다.
