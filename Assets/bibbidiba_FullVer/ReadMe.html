 ■モーションを使用される際の注意点


　・標準＋特殊ボーン(上半身2,グルーブ,肩P,腕捩,手捩,親指０)を使用しています


　・動作タイミングに合う音源
　　同梱したMovieから音源を抜き取った方が早いです。

　　もしご自分で調達する場合、ビビデバ公式MVの音源は使えません
　　理由は冒頭にセリフ「カット！」が入っているのでその部分をトリミングすればOKとは思いますが別ソフトが必要になるのでお勧めしません。


　・「まばたき」と「目線」はそれっぽく付けていますが、こまかな表情は一切付けていません
　　利用者側で使用する自作カメラモーションに合わせてがんばって付けてください。
　　理由：モーション利用者が自作カメラで撮影するであろうと想定しているから、細かく表情をつけすぎると利用者側での修正が多発する為、最低限に留めた。


　・リップシンクモーションについて
    bibbidibaFull_DanceMotionにはリップシンクモーションは入っていませんので、ダンスモーション投入後にbibbidibaFull_LipsyncMotionを投入してください。
    ※bibbidibaFull_LipsyncMotionはリップのみとなります。
